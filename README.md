🏢 UAE Real Estate Viewer

Десктопное приложение на Python для работы с API недвижимости (ОАЭ).
Позволяет получать список жилых комплексов, фильтровать их и просматривать данные по квартирам.
Проект демонстрирует навыки работы с REST API, обработкой данных и построением GUI-интерфейса.

🚀 Функциональность

Получение проектов через REST API
Фильтрация по названию
Просмотр информации по выбранному ЖК
Загрузка списка квартир (units) по проекту
Вывод данных в формате JSON
Постраничная загрузка данных (offset / limit)
Обработка ошибок API

🛠 Используемый стек

Python 3.11+
requests
tkinter (GUI)
JSON
REST API
Работа с параметрами запроса (pagination)
Обработка исключений

🧠 Что демонстрирует проект
Работа с внешним API
Авторизация через Bearer Token
Обработка больших массивов данных
Разделение логики (API / бизнес-логика / UI)
Работа с асинхронными запросами (через пагинацию)
Фильтрация и поиск по данным
Формирование пользовательского интерфейса
